CONTROLES:

Modo Camara (on/off): 
Shift + F

Si el modo cámara se encuentra activo:
	W -> mover adelante
	S -> mover atrás
	A -> mover izquierda
	D -> mover derecha
	Q -> mover abajo
	E -> mover arriba
	Movimiendo del mouse -> rota la cámara

Si el modo cámara está inactivo:
	Shift + Click izquierdo en modelo -> picking en modelo
